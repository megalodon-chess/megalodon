# ELO

The ELO rating system is a relative scoring system.
Please look at [this Wikipedia article][wiki] for more information.

## ELO Difference

These scores are relative to version 0.3.0

* v0.3.1: +133.3 ELO
* v0.3.2: +168.7 ELO
* v0.4.0: +188.2 ELO
